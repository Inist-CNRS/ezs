# trello for EZS plugin

This package cannot be used alone. EZS has to be installed.

## Example

```js
#!/usr/bin/env ezs

[use]
plugin = trello

[trello]
param = A
param = 1
```

## Installation

    $ git clone https://github.com/touv/node-trello.git
    $ cd node-trello
    $ npm install -g ezs
    $ npm install
    $ npm link
    $ npm run build 
    $ echo "trello do nothing !" | ./examples/trello.ezs
